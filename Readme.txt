Siin on siis 8h jagu tööd.
Toon kohe välja et, ei ole paljusid asju lõpetatud!
ES 6 - oli plaanis pärast seda kui kõik valmis. Kuna ma ei ole viimasel ajal JS ilma raamistikudena kasutanud siis on hetkel veel ES 6 õppimis järgus.
CSS ei ole ka responsiv veel 100% vaja teha tööd meediate kallal.
Kogu JS script asub hetkel loadXML.html failis. Reeglina valmis lahenduse puhul impordin JS sisse eraldi failist nt. app.js.
Hetkel genereerib tabelina andmed ilmajaama xmlist ja apiux xml käib lihtsalt läbi ja viimasele päringule.
Vaja on veel teha seal üsna palju:
	1. Error handling
	2. Loading - et ei kuvataks tühja infot ennem kui on olemas kogu info
	3. ES 6 kood kohantada vastavalt nõuetele
	4. ajax uuendamine mingi intervallitagant (ainult andmed)
	5. XML parse whitepace ja tühjad tagid 
	6. jne...
Kuna poolikut tööd on üsna keeruline vabantada ja seletada loodan, et saad mingi ülevaate.
Parimat!!
